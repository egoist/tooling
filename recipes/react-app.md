# Create React App

Create a new project, install dependencies.

```bash
$ tooling init my-project --react
```

Following the output instructions:

```bash
$ cd my-project
$ npm install
$ npm start
```

## Run it in development mode

```bash
$ npm start

# or with live reloading
$ tooling --live
```

## Build for production use

```bash
$ npm run build
```
